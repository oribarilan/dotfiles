# dotfiles

## Key Features

- <C-h/j/k/l> pane navigation in tmux & nvim
- <C-a> leader for tmux
- <C-a>F for tmux fzf (session management)
- <C-a>- split horizontally
- <C-a>\ split vertically
- <C-a>x kill pane
- <C-a>z zoom pane (default)
- <C-a>h/j/k/l resize current pane
